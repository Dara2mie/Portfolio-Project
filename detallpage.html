SmartBanking App Redesign

A clean, intuitive, and responsive banking experience designed to improve trust, usability, and user retention for Gen Z and Millennial users.

Role: UI/UX Design

Overview

This project aimed to modernize an outdated mobile banking UI. The client, a fintech startup, needed fresh UI and smoother onboarding experience to reduce user churn and increase trust.

Goals & Objectives
  • Simplify navigation flow
  • Improve dashboard responsiveness
  • Implement secure but user-friendly login flows
Tools & Tech Stack

Figma | React | Firebase Auth | TailwindCSS

Challenges

A key challenge was simplifying complex financial terms without losing meaning. I worked with a fintech writer to craft user-friendly microcopy and prioritized accessibility from the start, especially for screen readers and keyboard navigation.

Goals & Objectives

Looking to add biometric login and integrate cognitive tracking analytics in the next version.

Processes

Research & Insights

Conducted 15 user interviews with fintech users aged 22â€“35 to uncover key pain points and feature gaps.

Ideation & Design

Sketched low-fidelity wireframes, then built high-fidelity mockups in Figma with a clean, mobile-first layout.

Development

Built responsive UI in React using TailwindCSS and connected with Firebase and REST API.

Results

Ran usability tests with 7 users; 85% completed onboarding. Onboarding time dropped by 40%.
